Data Format

Each instance includes 5 fields as follows:

  • id: unique id for a receipt

  • image_path: image path associated with receipt

  • anno_quality_score: a manually annotated score based on the number of clear text lines.

  • anno_polygons: a manually annotated polygons of text fields.

  • anno_texts: a manually annotated text fields.

Training/Testing Data

Participants will be provided approximately 1,600 training examples with the respective annotated information. The testing set consists of 200 examples without annotations.
